\section{Purpose and scope}
Module 3 is the first post-radiation evolution model in the project. Module 1 tells us where radiation deposited energy and, after additional modeling assumptions, where defects were created. Module 3 answers the next question: once defects exist inside silicon, how do they move and transform with time?

The first implementation target is intentionally narrow and rigorous:
\begin{itemize}
    \item two spatial dimensions, \(x\) and \(y\), on a structured Cartesian grid,
    \item one defect concentration field \(C(x,y,t)\),
    \item diffusion of defects through the silicon,
    \item first-order annealing as the simplest reaction channel,
    \item zero-normal-flux boundary conditions for a closed-material baseline, and
    \item verification cases that isolate diffusion and annealing separately.
\end{itemize}

This note is written so that the physics and the MATLAB implementation stay tightly connected. The equations are stated first, then reduced to the specific numerical form we will actually solve.

\section{Physical picture}
A radiation event can displace atoms, create vacancies and interstitials, and generate more complicated defect structures. Immediately after the event, the damage is not yet a final material state. Defects can:
\begin{itemize}
    \item migrate through the lattice,
    \item recombine with complementary defects,
    \item form clusters,
    \item dissociate from larger complexes, or
    \item anneal away as the system moves toward a lower-energy configuration.
\end{itemize}

The full physics can become chemically and crystallographically rich. The first 2D solver deliberately starts below that complexity. It asks for the evolution of a coarse-grained defect concentration field, not the trajectory of each atom. In that coarse-grained description, the two most important mechanisms to represent first are:
\begin{enumerate}
    \item \textbf{Diffusion:} defects spread from regions of high concentration toward regions of lower concentration.
    \item \textbf{Annealing:} defects disappear from the tracked population at a rate controlled by local kinetics.
\end{enumerate}

That gives a model that is simple enough to verify analytically, but still physically meaningful as a bridge between the radiation source map and later device-level degradation calculations.

\section{Governing equation}
\subsection{Conservation statement}
Let \(C(x,y,t)\) denote the concentration of one defect species. For a small material area element, conservation requires
\begin{equation}
\pd{C}{t} = -\grad \cdot \bm{J} + R,
\label{eq:conservation}
\end{equation}
where \(\bm{J}\) is the defect flux vector and \(R\) is the net local reaction term.

Equation~\eqref{eq:conservation} is the cleanest starting point because it states the physics directly:
\begin{itemize}
    \item the concentration changes because defects flow in or out of a region, and/or
    \item the concentration changes because defects are created or removed by local reactions.
\end{itemize}

\subsection{Diffusive flux}
For the first implementation, we assume Fickian diffusion:
\begin{equation}
\bm{J} = -D\,\grad C,
\label{eq:fick}
\end{equation}
where \(D\) is the defect diffusion coefficient.

Substituting Equation~\eqref{eq:fick} into Equation~\eqref{eq:conservation} gives
\begin{equation}
\pd{C}{t} = \grad\cdot\left(D\,\grad C\right) + R.
\label{eq:general_diffusion_reaction}
\end{equation}

If \(D\) is spatially uniform, this reduces to
\begin{equation}
\pd{C}{t} = D\lap C + R.
\label{eq:uniformD}
\end{equation}

In 2D Cartesian coordinates,
\begin{equation}
\pd{C}{t} = \frac{\partial}{\partial x}\left(D\pd{C}{x}\right) + \frac{\partial}{\partial y}\left(D\pd{C}{y}\right) + R.
\label{eq:2d_general}
\end{equation}

For constant \(D\),
\begin{equation}
\pd{C}{t} = D\left(\pdd{C}{x}+\pdd{C}{y}\right) + R.
\label{eq:2d_constantD}
\end{equation}

\subsection{First-order annealing model}
The first reaction model is first-order annealing:
\begin{equation}
R = -k_{\mathrm{ann}} C,
\label{eq:first_order_ann}
\end{equation}
where \(k_{\mathrm{ann}}\) is the annealing rate constant.

Then the governing equation becomes
\begin{equation}
\boxed{\pd{C}{t} = D\left(\pdd{C}{x}+\pdd{C}{y}\right) - k_{\mathrm{ann}} C.}
\label{eq:final_pde}
\end{equation}

This is the baseline PDE for the first 2D Module 3 implementation.

\section{Why this model is the right first step}
This first PDE is not the final defect-chemistry model. It is the correct first step because it gives all of the following at once:
\begin{itemize}
    \item a direct way to consume a 2D damage map from Module 1,
    \item a physically meaningful time evolution of damage,
    \item a field \(C(x,y,t)\) that later modules can map into charge, recombination, and thermal effects,
    \item analytic limiting cases for verification, and
    \item a clean extension path to multiple defect species \(C_i\), nonlinear reactions \(R_i\), and temperature-dependent kinetics.
\end{itemize}

In other words, Equation~\eqref{eq:final_pde} is simple enough to debug and strong enough to become infrastructure.

\section{Boundary and initial conditions}
\subsection{Boundary conditions}
The first implementation uses zero-normal-flux boundary conditions:
\begin{equation}
\bm{n}\cdot\bm{J}=0,
\end{equation}
where \(\bm{n}\) is the outward unit normal vector at the boundary.

Using \(\bm{J}=-D\grad C\), this becomes
\begin{equation}
\bm{n}\cdot\grad C=0.
\label{eq:neumann_bc}
\end{equation}

Physically, this means defects do not leave the computational domain through the outer boundary in the baseline case.

\subsection{Initial condition}
The initial condition can come from either a synthetic verification profile or a processed Geant4-derived map:
\begin{equation}
C(x,y,0)=C_0(x,y).
\end{equation}

For verification, two convenient choices are:
\begin{itemize}
    \item a 2D Gaussian cloud, and
    \item a spatially uniform field.
\end{itemize}

\section{Analytic limiting cases for verification}
\subsection{Pure annealing}
If diffusion is switched off, so that \(D=0\), Equation~\eqref{eq:final_pde} reduces to
\begin{equation}
\pd{C}{t} = -k_{\mathrm{ann}} C.
\end{equation}

At every spatial point, the solution is
\begin{equation}
C(x,y,t)=C(x,y,0)\,e^{-k_{\mathrm{ann}}t}.
\label{eq:pure_anneal_solution}
\end{equation}

If the initial field is uniform, the field stays uniform for all time and decays exponentially. This is an excellent code test because any spatial structure that appears is numerical error.

\subsection{Pure diffusion}
If annealing is switched off, so that \(k_{\mathrm{ann}}=0\), Equation~\eqref{eq:final_pde} becomes
\begin{equation}
\pd{C}{t} = D\lap C.
\end{equation}

For a localized initial cloud in an effectively large domain, the defect distribution broadens with time while conserving total defect amount,
\begin{equation}
M(t)=\iint C(x,y,t)\,\dd x\,\dd y = \text{constant},
\label{eq:mass_conservation}
\end{equation}
provided the boundary flux is zero.

A Gaussian profile remains Gaussian under pure diffusion, with a width that grows in time. Even if the finite computational domain and boundaries modify the exact analytic shape, the qualitative signatures are robust:
\begin{itemize}
    \item peak concentration decreases,
    \item cloud width increases,
    \item total mass remains nearly constant.
\end{itemize}

\section{Temperature dependence and future coupling}
Even though the first version may hold temperature fixed, Module 3 is fundamentally temperature sensitive. In many defect processes, both the diffusion coefficient and the annealing rate are thermally activated. A common form is Arrhenius-like behavior:
\begin{align}
D(T) &= D_0\exp\!\left(-\frac{E_D}{k_B T}\right), \\
 k_{\mathrm{ann}}(T) &= k_0\exp\!\left(-\frac{E_a}{k_B T}\right),
\end{align}
where \(D_0\) and \(k_0\) are prefactors, \(E_D\) is a diffusion activation energy, \(E_a\) is an annealing activation energy, and \(k_B\) is Boltzmann's constant.

This matters for the project roadmap because Module 4 will provide the temperature field \(T(x,y,t)\). Once that happens, Module 3 can be upgraded from constant-parameter kinetics to locally temperature-dependent kinetics without changing the overall conservation form of the solver.

\section{Numerical implementation strategy}
\subsection{Why a structured 2D Cartesian baseline}
The project-level roadmap refers to diffusion-reaction finite element modeling, which is a reasonable long-term direction for arbitrary geometries and heterogeneous materials. For the first implementation, however, a structured 2D Cartesian discretization is the better engineering choice because it is:
\begin{itemize}
    \item easier to implement correctly,
    \item easier to verify against analytic limits,
    \item easier to visualize and debug, and
    \item fully adequate for the first rectangular silicon-domain studies.
\end{itemize}

So the immediate implementation should be understood as a \textbf{2D structured baseline} that establishes the physics and software interface before a more general discretization is introduced.

\subsection{Grid}
Let the computational domain be discretized by \(N_x\) points in the \(x\)-direction and \(N_y\) points in the \(y\)-direction, with spacings
\begin{equation}
\Delta x = x_{i+1}-x_i,
\qquad
\Delta y = y_{j+1}-y_j.
\end{equation}

The discrete approximation to the concentration field is written as \(C_{i,j}^n\), meaning the value at grid location \((i,j)\) and time step \(n\).

\subsection{Spatial discretization}
For uniform \(D\), the standard second-order central-difference Laplacian is
\begin{equation}
\lap C\big|_{i,j} \approx
\frac{C_{i+1,j}-2C_{i,j}+C_{i-1,j}}{\Delta x^2}
+
\frac{C_{i,j+1}-2C_{i,j}+C_{i,j-1}}{\Delta y^2}.
\label{eq:lap_discrete}
\end{equation}

With mirrored ghost-cell logic for the Neumann boundary condition, the derivative normal to the boundary is forced to zero.

\subsection{Time stepping}
The first implementation can use an explicit forward-Euler update,
\begin{equation}
C_{i,j}^{n+1} = C_{i,j}^{n} + \Delta t\left[D\,\lap_h C_{i,j}^{n} - k_{\mathrm{ann}} C_{i,j}^{n}\right],
\label{eq:explicit_update}
\end{equation}
where \(\lap_h\) denotes the discrete Laplacian.

This is straightforward and good for verification, but it carries a diffusion stability restriction. A practical estimate is
\begin{equation}
\Delta t \lesssim \frac{1}{2D\left(\frac{1}{\Delta x^2}+\frac{1}{\Delta y^2}\right)}.
\label{eq:dt_limit}
\end{equation}

Equation~\eqref{eq:dt_limit} is crucial. When the grid is refined, the stable explicit time step decreases rapidly. That is acceptable for the first verification stage, but it is also the reason the project should later consider:
\begin{itemize}
    \item implicit diffusion solves,
    \item operator splitting, or
    \item sparse linear-system formulations.
\end{itemize}

\subsection{Mass and positivity checks}
For the pure-diffusion case with zero-flux boundaries, the discrete total mass should remain approximately constant:
\begin{equation}
M^n = \sum_{i,j} C_{i,j}^n\,\Delta x\,\Delta y.
\end{equation}

For the pure-annealing case, the spatially uniform initial state should remain uniform and follow Equation~\eqref{eq:pure_anneal_solution}. In practice, it is also reasonable to clamp tiny negative values caused by explicit time stepping and roundoff, but that should be treated as a numerical safeguard, not as a substitute for a stable update scheme.

\section{Verification plan}
The first 2D Module 3 implementation should not begin with full Geant4-coupled runs. It should begin with controlled tests.

\subsection{Verification Case A: 2D Gaussian diffusion}
\textbf{Setup:}
\begin{itemize}
    \item initial condition: localized 2D Gaussian,
    \item diffusion coefficient \(D>0\),
    \item annealing rate \(k_{\mathrm{ann}}=0\),
    \item zero-flux boundaries.
\end{itemize}

\textbf{Expected behavior:}
\begin{itemize}
    \item the peak concentration decreases,
    \item the profile broadens,
    \item the total defect mass remains nearly constant.
\end{itemize}

\subsection{Verification Case B: uniform-state preservation}
\textbf{Setup:}
\begin{itemize}
    \item initial condition: \(C(x,y,0)=C_0\),
    \item diffusion coefficient \(D>0\),
    \item annealing rate \(k_{\mathrm{ann}}=0\).
\end{itemize}

\textbf{Expected behavior:}
\begin{itemize}
    \item the field remains uniform to numerical precision.
\end{itemize}

This test is especially useful for catching boundary-condition mistakes or indexing bugs in the discrete Laplacian.

\subsection{Verification Case C: pure annealing}
\textbf{Setup:}
\begin{itemize}
    \item initial condition: uniform or nonuniform,
    \item diffusion coefficient \(D=0\),
    \item annealing rate \(k_{\mathrm{ann}}>0\).
\end{itemize}

\textbf{Expected behavior:}
\begin{itemize}
    \item every grid point decays as \(e^{-k_{\mathrm{ann}}t}\),
    \item a uniform field stays uniform,
    \item numerical results can be directly compared against Equation~\eqref{eq:pure_anneal_solution}.
\end{itemize}

\subsection{Verification Case D: imported damage map}
Only after Cases A-C are behaving correctly should the code ingest a processed 2D damage map derived from Module 1. At that stage, the key checks are not analytic exactness, but rather:
\begin{itemize}
    \item consistent data import,
    \item sensible physical smoothing and decay,
    \item reproducible saved history, and
    \item readiness for coupling to Module 2 and Module 4.
\end{itemize}

\section{How Module 3 connects to the rest of the project}
Module 3 is not an isolated math exercise. Its output feeds directly into later modules:
\begin{itemize}
    \item \textbf{Module 2:} defect concentration can be mapped into defect charge or trap density, which changes the space-charge field and electrostatic potential.
    \item \textbf{Module 4:} temperature modifies diffusion and annealing rates, so the thermal solver supplies \(T(x,y,t)\) back into Module 3.
    \item \textbf{Module 5:} evolving defect populations alter recombination, trapping, and carrier lifetime, which changes electrical response.
    \item \textbf{Module 6:} the full coupled solver iterates among these effects self-consistently.
\end{itemize}

That is why building Module 3 first is strategically correct. It establishes the first dynamic material-state variable in the MATLAB side of the project.

\section{Planned extensions beyond the first solver}
Once the baseline 2D single-species solver is working and verified, the next upgrades are natural:
\begin{enumerate}
    \item multiple defect species \(C_i\),
    \item nonlinear reaction channels \(R_i(C_1,C_2,\ldots)\),
    \item temperature-dependent \(D(T)\) and \(k_{\mathrm{ann}}(T)\),
    \item spatially varying material parameters,
    \item implicit or semi-implicit time integration, and
    \item more general geometries and discretizations.
\end{enumerate}

In short, the first 2D solver is not a detour. It is the foundation on which the richer defect-physics model will be built.
